# CSS

## 概述

### 基础

1. **CSS**: ( Casscading Style Sheet)，译名为层叠样式表， 是一门样式表语言。
2. **规则集**:一个样式设计由选择器和规则集组成，规则集包含众多的声明，声明由属性和值组成的键值对组成。
3. **选择器**:标签选择器， ID选择器， 类选择器。
4. **选择器修饰**: 属性 => `input[type=text]`; 伪类=> `a:hover`。
