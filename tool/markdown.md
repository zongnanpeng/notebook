# markdown
## 概述
### html
>html标签大部分可用  

>特殊字符转换，因此 &lt; 和 & 标签需要实体形式，'&lt'; 和'&amp;';
## 区块元素
### 段落
> 用enter产生一个空行或者用 p 标签。
### 换行
>用两个空格或者br 标签。
### 标题
>用 ‘#’的个数代表1-6级标题
### 区块
>用 &gt; 的个数表示区块的层级。
### 代码区块
> 一个制表符。
### 无序列表
>用* | + | - 都可以达到相同的效果。
### 有序列表
> 用数字加点号表示。
### 分割线
> 用三个 * | - | _ 。

## 段落元素
### 链接
>中括号内为链接名，小括号内为链接地址。  
eg: [百度](https://www.baidu.com)
### 强调
>用两个星号包括。  
eg: **强调加粗字体**
### 斜体
>用 一对星号包括  
eg: *斜体字体*
### 代码
>反引号  
eg： `println("hello world")`
